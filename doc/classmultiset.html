Detailed Description

template<typename T, typename funct>
class multiset< T, funct >

Classe che implementa un mutliset. I dati base sono coppie <elemento, occorrenze>. La classe memorizza tali coppie attraverso una lista. La classe richiede la specifica del tipo di elemento T e del funtore funct per la comparazione di due elementi.
